##### 解构赋值

1、数组的结构

```
const F4 = ['小沈阳','刘能','赵四','宋小宝'];

```

2、对象的结构

```javascript
const obj = {
  s: "123",
  ss: "323",
};
let { s, ss } = obj;
console.log(s);
console.log(ss);
```

##### 箭头函数

```javascript
// 声明函数
let fn = function () {};
// 箭头函数
let fn2 = () => {};
// this 是静态的，this 始终指向函数声明时所在作用域下的this的值
// 不能作为构造实例化对象
// 不能使用arguments变量
//箭头函数的简写
// 1、省略小括号，当形参有且只有一个的时候
// 2、省略花括号，当代码体有且只有一个
// 箭头函数适合于this无关的回调，定时器，数组方法的回调
// 箭头函数不适合与this有关的回调，如事件回调，对象的方法
```

##### 参数默认值

```javascript
function add(a, b, c = 10) {
  return a + b + c;
}
// 与解构赋值结合
function conect({ username, password }) {
  console.log(username);
  console.log(password);
}
```

##### rest 参数，用于获取函数的实参，用来代替 arguments

```javascript
function date(...args) {
  console.log(args);
}
date("1", "2", "3"); // [1,2,3]
// rest参数必须要放到参数最后
```
